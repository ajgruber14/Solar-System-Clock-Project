This solar system clock was designed by myself primarily using HTML and CSS. The Javascript portion of this project is used to synchronize the clock with the client's relative time and provide an accurate rate of rotation for each planet.

Credit is due to Web Dev Simplified for providing a technique to model time values from a date object in 360 degrees.

note: Some of the information and terms used in the breakdown below may contain inaccuracies as this is a novice interpretation of js syntax. If you notice any errors or would like to provide further clarification, feel free to reach out and/or correct me!

Javascript

const seconds = document.querySelector('[data-seconds]');
const minutes = document.querySelector('[data-minutes]');
const hours   = document.querySelector('[data-hours]');
        

setInterval(setClock, 1000);

function setClock() {
    const currentDate  = new Date;
    const secondsRatio = currentDate.getSeconds() / 60;
    const minutesRatio = (secondsRatio + currentDate.getMinutes()) / 60;
    const hoursRatio   = (minutesRatio + currentDate.getHours()) / 12;

    setRotation(seconds, secondsRatio);
    setRotation(minutes, minutesRatio);
    setRotation(hours,   hoursRatio );
}
        

function setRotation(element, rotationRatio) {
    element.style.setProperty('--rotation', rotationRatio * 360);
}
        

setClock();

setClock() is the final piece to the project. It initiates the setClock function upon loading!
